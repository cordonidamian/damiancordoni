#3 Python Certification - How did I get a 50% discount for the PCAP exam?

I will be sharing a interesting option to learn Python and be prepared to get a certification March 25, 2020

As I mentioned in the intro I clicked on this link... there you will find the presentation for the The PCAP Programming Essentials in Python Course and the opportunity to enroll, for now it´s free.

Cisco Networking Academy in collaboration with OpenEDG Python Institute are giving us the chance to learn Python with this course.

It is divided in two block:

  • Basics
    • Introduction to Python and computer programming
    • Data types, variables, basic input-output operations, basic operators
    • Boolean values, conditional execution, loops, lists and list processing, logical and bitwise operations
    • Module 4 - Functions, tuples, dictionaries, and data processing
  • Intermediate
    • Modules, packages, string and list methods, and exceptions
    • The object-oriented approach: classes, methods, objects, and the standard objective features; exception handling, and working with files

Each module has multiple exercises, examples and each block has a test. The first block (Basics) gives the knowledge to be ready for the Entry-Level Python Programmer Certification and once you complete the second block (Intermediate) and the final test course you will be able to take the Associate in Python Programming certification. That´s not all! If you pass the course, you will receive a 50% discount for a voucher.
